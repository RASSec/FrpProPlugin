# FrpProPlugin
 frp0.33修改版,过流量检测,免杀,支持加载远程配置文件可用于cs直接使用的插件。需要服务端适配，可以在release下载对应平台的服务端。

## 更新时间线

2021.2.19 免杀更新+缩小体积
